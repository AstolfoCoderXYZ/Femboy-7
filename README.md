# Femboy 7 Ultimate — Windows Vista Simulator

Este repositorio contiene una simulación web interactiva de un escritorio inspirado en Windows Vista, con una temática personalizada denominada **Femboy 7 Ultimate**. El proyecto recrea la experiencia visual de Windows Aero con un toque estético único.

## 🚀 Características

- **Pantalla de Bloqueo:** Interfaz de inicio de sesión funcional con efectos de transición.
- **Efectos Aero Glass:** Ventanas con transparencias, desenfoque (backdrop-filter) y reflejos realistas.
- **Gestión de Ventanas:**
  - Ventanas arrastrables y redimensionables.
  - Funcionalidad de maximizar, minimizar y cerrar.
  - Vistas previas en la barra de tareas al pasar el ratón.
- **Aplicaciones Incluidas:**
  - **Equipo / Explorador de archivos:** Navegación simulada.
  - **Femboy Media Player:** Reproductor de música que soporta arrastrar y soltar archivos locales (.mp3, etc.) y lectura de metadatos (ID3).
  - **Femboy Messenger:** Chat simulado con respuestas automáticas.
  - **Centro de OwO:** Monitor de estado del sistema (seguridad, firewall, etc.).
  - **Administrador de Tareas:** Visualización de procesos ficticios y uso de recursos en tiempo real.
  - **Sticky Notes:** Notas adhesivas persistentes (se guardan en el navegador).
  - **Personalización:** Cambio dinámico de colores Aero (Rosa, Lavanda, Medianoche, Menta).
- **Gadgets de Escritorio:** Reloj analógico/digital, medidores de rendimiento (CPU/RAM) y estado de red en una barra lateral.
- **Salvapantallas:** Activación automática tras periodo de inactividad con efecto de estrellas.

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructura semántica y contenedores.
- **CSS3:** Estilización avanzada, animaciones, Flexbox/Grid y efectos de desenfoque Aero Glass.
- **JavaScript (Vanilla):** Lógica de manipulación del DOM, gestión de ventanas, simulación de procesos y persistencia de datos.
- **jsmediatags:** Librería externa para extraer metadatos de archivos de audio.
- **Google Fonts:** Tipografía "Segoe UI" para mayor fidelidad al sistema original.

## 📂 Estructura del Proyecto

- `index.html`: Punto de entrada principal y estructura del escritorio.
- `style.css`: Definición de estilos, temas de color y efectos visuales.
- `script.js`: Toda la lógica interactiva del sistema.
- `icons/`: Directorio que contiene los iconos y recursos gráficos utilizados.
- `hlyxrw.mp3`: Sonido de inicio de sesión.

## 💻 Cómo Ejecutarlo

1. Clona este repositorio o descarga los archivos.
2. Abre el archivo `index.html` en cualquier navegador web moderno (Chrome, Firefox, Edge, etc.).
3. ¡Disfruta de la experiencia!
